Tentang HTML

Halo ini adalah halaman pertama saya dalam belajar html. HTML (HyperText Markup Language) merupakan bahasa pemrograman yang digunakan untuk membuat struktur dan konten halaman web.

HTML terdiri dari tag-tag yang memberikan instruksi kepada browser untuk menampilkan konten yang diinginkan. Tag-tag tersebut diletakkan di dalam tanda kurung siku dan biasanya berpasangan, yaitu tag pembuka dan tag penutup.

Contoh tag pembuka dan tag penutup adalah <p> dan </p>. Tag pembuka digunakan untuk memulai suatu elemen, sedangkan tag penutup digunakan untuk mengakhiri suatu elemen.

Contoh lain tag pembuka dan tag penutup adalah <h1> dan </h1>. Tag <h1> digunakan untuk membuat heading level 1, sedangkan tag </h1> digunakan untuk mengakhiri heading level 1.

Struktur Dasar HTML

Struktur dasar HTML terdiri dari tag <!DOCTYPE html>, <html>, <head>, dan <body>.

Tag <!DOCTYPE html> digunakan untuk menentukan versi HTML yang digunakan. Tag <html> digunakan untuk membungkus seluruh elemen HTML. Tag <head> digunakan untuk menyimpan informasi tentang dokumen HTML. Tag <body> digunakan untuk menyimpan konten yang akan ditampilkan di halaman web.

Berikut Contohnya 

            
                <!DOCTYPE html>
                <html>
                    <head>
                        <title>Judul Halaman</title>
                    </head>
                    <body>
                        <h1>Heading Level 1</h1>
                        <p>Paragraf pertama</p>
                        <p>Paragraf kedua</p>
                    </body>
                </html>

span dan div

Selain tag pembuka dan tag penutup, terdapat juga tag yang tidak memerlukan tag penutup, yaitu tag <span> dan <div>. Tag <span> digunakan untuk memberikan gaya pada sebagian teks, sedangkan tag <div> digunakan untuk membungkus beberapa elemen HTML.

Berikut contoh dari span dan div 

            
                <p>Ini adalah contoh teks <span style="color: red;">span</span></p>
                <div style="background-color: lightblue;">
                    <p>Paragraf pertama</p>
                    <p>Paragraf kedua</p>
                </div>

Element hr br sup sub

Element <hr> digunakan untuk membuat garis horizontal. Element <br> digunakan untuk memberikan jarak satu baris. Element <sup> digunakan untuk membuat teks menjadi superscript. Element <sub> digunakan untuk membuat teks menjadi subscript.
